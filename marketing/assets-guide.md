# Social Media Assets Guide

## 이미지 요구사항

### Product Hunt
- **Thumbnail:** 240x240px
- **Gallery Images:** 1270x760px (최대 8장)
- **형식:** PNG 또는 JPG

**필수 이미지:**
1. **Hero Image** (1270x760)
   - VibeStack 메인 페이지 스크린샷
   - "50+ AI Tools" 텍스트 포함
   
2. **Ranking Page** (1270x760)
   - Top 10 rankings screenshot
   - "Best AI Coding Tools 2025"

3. **Tool Detail** (1270x760)
   - 개별 도구 페이지
   - Cursor 또는 Claude 페이지

### Twitter/X
- **Card Image:** 1200x675px (16:9)
- **Profile Banner:** 1500x500px

### Reddit
- **Thumbnail:** 자동 생성됨
- **Link Preview:** Open Graph 메타데이터 사용

### LinkedIn
- **Image:** 1200x627px
- **Article Cover:** 1200x644px

## Canva 템플릿 제안

**브랜드 컬러:**
- Primary: #6366f1 (Indigo)
- Secondary: #8b5cf6 (Purple)
- Background: #0f172a (Dark)
- Text: #f8fafc (White)

**템플릿 구조:**
```
[Logo]
   
[Main Visual]
Tool Icons Grid or Screenshot

[Headline]
"Top 10 AI Coding Tools 2025"
   
[CTA]
usevibestack.com
```

## 스크린샷 촬영 가이드

**Mac:**
```bash
# 전체 화면
Cmd + Shift + 3

# 선택 영역  
Cmd + Shift + 4

# 특정 창
Cmd + Shift + 4, then Space
```

**촬영할 페이지:**
1. https://usevibestack.com (메인)
2. https://usevibestack.com/best-ai-coding-tools-2025 (랭킹)
3. https://usevibestack.com/tool/cursor (도구 상세)
4. https://usevibestack.com/stacks (스택 페이지)

**브라우저 설정:**
- 크롬 개발자 도구 (F12)
- Device Toolbar 선택
- 1440x900 해상도
- Dark mode 활성화
- 스크롤바 제거
